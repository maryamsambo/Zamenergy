# ⚡ ZamEnergy

ZamEnergy is a modern, responsive web application that promotes clean and sustainable energy solutions. It helps users explore solar energy options, estimate their energy needs, and learn about the benefits of renewable energy through an intuitive and user-friendly interface.

## Features

- Responsive design
- Solar energy information
- Energy estimation tools
- Fast and modern UI
- Mobile-friendly experience

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

Clone the repository:

```bash
git clone https://github.com/maryamsambo/Zamenergy.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
